# RedCart

This repository hosts *RedCart*, a beginner-level React project that I built while learning the basics of React. It represents a simple e-commerce shopping cart interface with a focus on fundamental concepts like components, state, props, and styling.

## ?? Purpose

I created RedCart as part of my journey into React development. The goal was to get hands-on experience building small, interactive UI elements and to reinforce my understanding of React's core principles.

## ?? Features & Functionality

- **Component-based structure**: The app is organized into reusable functional components such as Navbar, Aside, Card, Container, MainPage, and Products.
- **State management with hooks**: Uses useState (and optionally useEffect) to manage cart items and UI interactions.
- **Responsive layout**: Basic responsive design using CSS modules placed in the styles/ directory to ensure the interface looks good on different screen sizes.
- **Interactive cart functionality**: Users can view products, add items to the cart, and see updates in the Aside component.
- **Styling with plain CSS**: Simple styles are defined in separate CSS files under src/styles/ for clarity and ease of learning.

## ?? How to Run

1. Clone the repository.
2. Navigate into the project folder: cd RedCart.
3. Install dependencies: 
pm install (or yarn).
4. Start the development server: 
pm run dev.
