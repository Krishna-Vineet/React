

## 🖤 **Dark E-Commerce Theme – Color Palette by Component**

| **Component**       | **Color**      | **Hex Code** | **Purpose / Description**         |
| ------------------- | -------------- | ------------ | --------------------------------- |
| **Background**      | Charcoal Black | `#0E0E10`    | Overall app/page background       |
| **Card Background** | Deep Graphite  | `#1A1A1D`    | Product cards, modals, containers |
| **Navbar**          | Jet Black      | `#141416`    | Fixed top navigation bar          |
| **Sidebar**         | Carbon Gray    | `#171719`    | Left/right drawer or menu         |
| **Primary Text**    | Pure White     | `#FFFFFF`    | Main readable content             |
| **Secondary Text**  | Soft Gray      | `#B3B3B3`    | Descriptions, subtext, labels     |
| **Heading Text**    | Platinum White | `#F5F5F7`    | H1–H3 style bold headings         |
| **Icons**           | Cool Gray      | `#CCCCCC`    | Static icons                      |
| **Primary Accent**  | Coral Rose     | `#FD4E6D`    | Buttons, CTAs, highlights         |
| **Hover Effect**    | Glow Pink      | `#FF4F6E`    | On hover for buttons/links        |
| **Border/Divider**  | Gunmetal       | `#2C2C2E`    | Subtle UI element separation      |
| **Success State**   | Mint Green     | `#4ADE80`    | For success badges, banners       |
| **Warning State**   | Amber Yellow   | `#FBBF24`    | For stock alerts, tips            |
| **Error State**     | Bright Red     | `#EF4444`    | Errors, validation, alerts        |

## ✨ **UI Effects & Styling Suggestions**

| **Element**     | **Style Suggestion**                                                    |
| --------------- | ----------------------------------------------------------------------- |
| **Buttons**     | Background: `#E94560`, text: `#FFFFFF`, hover: slight scale + `#FF4F6E` |
| **Cards**       | Rounded corners (`border-radius: 1rem`), inner shadow, light border     |
| **Navbar**      | Slight drop shadow + blur (`backdrop-filter: blur(8px)`)                |
| **Sidebar**     | Slide-in, `box-shadow: -4px 0px 16px rgba(0,0,0,0.5)`                   |
| **Links Hover** | Underline + color transition to `#079F91`                               |
| **Transitions** | `transition: all 0.3s ease-in-out` for smooth UX                        |
| **Box Shadows** | Use dark glowing shadows: `0px 4px 12px rgba(255, 79, 110, 0.2)`        |



| **Use Case**             | **Recommended Unit** |
| ------------------------ | -------------------- |
| Font sizes               | `rem`                |
| Spacing (padding/margin) | `rem` or `em`        |
| Border width, icons      | `px`                 |
| Layout widths            | `%` or `vw`          |
| Full-page sections       | `vh`, `svh`, `dvh`   |
| Text within components   | `em`                 |
| Paragraph width limit    | `ch`                 |
